BeArt Artverse

BeArt TEEChain is a high-performance public chain based on asynchronous consensus and hybrid proof mechanisms. It features both responsive and proactive smart contracts to meet developers’ diverse needs. Built on this chain, combined with cloud rendering technology, BeArt TEEChain provides underlying technical support and asset authentication for Artverse.
